# Simple Journal App - TypeScript Vue

## How to Install

```sh
npx degit francmarin98/vue-simple-journal-ts
```

```sh
npm install
npm run dev
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
